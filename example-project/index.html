Інтелектуальний додаток-наставник для йоги та фітнесу

Огляд проєкту

Над проектом працювали Камалова Дар'я та Королецька Софія, учениці 10-Г класу.

Камера аналізує твої рухи під час йоги та пілатесу. Система порівнює твоє положення тіла з основною позою та дає поради.

Мета проєкту:

Створити веб-застосунок, який аналізує рухи користувача в реальному часі через камеру, розпізнає пози тіла та дає зворотній зв'язок щодо правильності виконання вправ.

       

Це допоможе людям тренуватись вдома без травм.

Основні завдання:

  • Реалізація захоплення відео з камери в реальному часі
  • Зворотній зв'язок у реальному часі
  • Розпізнавання пози тіла
  • Створення зручного веб-інтерфейсу
  • Режим тренувань

Технічна реалізація

Використані технології:

  • Teachable Machine
  • GitHub

Інноваційні аспекти:

  • Оптимізований алгоритм обробки відеопотоку
  • Адаптивна система освітлення для покращення точності

Використані дані

  • Teachable Machine (Google)
  • GitHub
  • Власний набір даних для тестування (100 зображень)

Всі дані використовуються з дотриманням ліцензій та правил академічної доброчесності.

Результати

  • Точність розпізнавання поз: 89%
  • Швидкість обробки: 30 кадрів/сек
  • Підтримка 3 базових поз
