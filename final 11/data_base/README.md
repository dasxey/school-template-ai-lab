В цій папці буде знаходитись ... ..

id — унікальний номер запису

user_id — ідентифікатор користувача (для майбутнього розширення на багатьох користувачів)

date — дата та час запису

text_input — текстовий запис або транскрипт голосу

emotion — визначена емоція

emotion_score — оцінка інтенсивності емоції (0–100%)

gpt_answer — відповідь системи

Таблиця користувачів (users) (для розширеної версії)

user_id — унікальний ідентифікатор користувача

username — ім’я користувача (може бути анонімне)

registration_date — дата реєстрації
